# Spotify Search Application

A web application that allows users to search Spotify artists, albums, and tracks through a responsive frontend and backend API server.

This project was developed based on course starter code, with application features completed independently.

## Features

* Search for artists, albums, and tracks using Spotify data
* View detailed artist pages
* View album information and track lists
* Navigate between search results and detail pages
* Communicate between frontend client and backend web server

## Technologies

* Angular
* TypeScript
* Node.js
* Express
* Spotify API
* HTML / CSS

## Running the Project

Install dependencies for the client:

```bash
cd client
npm install
```

Install dependencies for the web server:

```bash
cd ../webserver
npm install
```

Start the backend server:

```bash
npm start
```

Start the frontend client in a separate terminal:

```bash
cd ../client
npm start
```
